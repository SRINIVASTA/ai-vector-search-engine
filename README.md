# 🚀 AI Search & WhatsApp Broadcast Suite

An embedding-powered semantic matching dashboard designed to parse recruitment listings using Google Generative AI, summarize contextual insights locally or via Groq LPU acceleration, and broadcast high-scannability notification cards directly to WhatsApp Community channels.

---
🌐 **Live Interactive Web App:** [Launch Live Streamlit Dashboard](https://ai-vector-search-engine-dptxoahhb8fewsk3h3nsjk.streamlit.app/)
---
## 🎯 Key Features

* **AI-Powered Semantic Engine**: Uses Google's `gemini-embedding-001` to map search intentions conceptually beyond rigid keyword parameters.
* **Dual-Reasoning Architecture**: Allows instant switching between an offline local clean text formatter and Groq’s high-speed cloud chip (Llama 3).
* **Zero-Quota Storage Design**: Eradicates background server state vulnerabilities by evaluating all credentials via transient in-memory sidebar inputs.
* **High-Scannability Card Layout**: Compresses search results into bulleted mobile notifications containing direct source paths, portal fields, and relative dynamic timeline metrics (`Today`, `1 day ago`).
* **Instant Community Integration**: Dispatches structured notification cards directly over the Green API text gateway platform.

## 📋 Technical Prerequisites

Ensure your environment framework contains an operating system with Python 3.10+ installed alongside active third-party account tokens from:
1. **Google AI Studio**: Required to process vector database coordinates.
2. **Groq Cloud (Optional)**: Required if using accelerated Llama 3 summarization models.
3. **Green API Developer Console**: Required to link your phone number and obtain channel routing targets.

## ⚙️ Local Machine Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com
   cd ai-vector-search-engine
   ```

2. **Initialize a Isolated Python Virtual Environment**
   ```bash
   python -m venv venv
   # On macOS / Linux:
   source venv/bin/activate
   # On Windows (Command Prompt):
   .\venv\Scripts\activate
   ```

3. **Install Dependencies**
   Create a `requirements.txt` file containing the application libraries and execute:
   ```bash
   pip install -r requirements.txt
   ```

4. **Launch the Local Web Application Server**
   ```bash
   streamlit run app.py
   ```

## 🚀 Running Your First Broadcast Pipeline

1. Open your browser dashboard path generated by Streamlit (typically `http://localhost:8501`).
2. Input your private validation key arrays into the secure sidebar input mask fields.
3. Type a targeted conceptual query inside the **🔍 Context Match Finder** (e.g., `data scientist, visakhapatnam`).
4. Review the structured output text preview box rendered on your center screen canvas.
5. Provide your WhatsApp destination credentials on the left side panel menu layout.
6. Click **🚀 Push Message to WhatsApp Community** to instantly transmit the job notification card live!
